# AM Materials Explorer

ANSYS Granta MI 스타일의 적층제조(AM) 소재 데이터베이스 탐색기

## GitHub Pages 배포 방법
1. 이 저장소 Settings → Pages
2. Source: Deploy from a branch → Branch: main / (root)
3. Save → 몇 분 후 https://{username}.github.io/{repo-name}/ 에서 접근 가능

## 기술 스택
React 19 + TypeScript + Tailwind CSS 4 + Recharts
